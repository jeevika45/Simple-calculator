 🧮 Python Simple Calculator

This project is a basic calculator built using Python.  
It allows users to perform simple arithmetic operations by entering numbers and selecting an operator.

## Features
- Addition (+)
- Subtraction (-)
- Multiplication (*)
- Division (/)

## How to Run
1. Download or clone the repository
2. Open the project folder
3. Run the program using:

python calculator.py

## Purpose
This project was created to practice Python basics such as user input, conditional statements, and arithmetic operations.
